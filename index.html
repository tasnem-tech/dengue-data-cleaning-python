🦟 Dengue Data Cleaning Project

Python Data Cleaning & Validation using Pandas

📌 Project Overview

This project focuses on cleaning and validating dengue outbreak datasets using Python and Pandas. The objective is to improve data quality, remove inconsistencies, detect anomalies, and prepare datasets for accurate analysis and reporting.

🚀 Key Tasks

  • Missing Value Analysis
  • Duplicate Detection
  • Standardizing Categorical Labels
  • Data Type Conversion
  • Outlier Detection (IQR Method)
  • Data Validation & Auditing
